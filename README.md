## Purpose

organize media files in given `./folder/` into

- `<folder>/media/`
  - `image/` for all image formats
  - `gif/` for all gifs
  - `video/` for a all video files

If the media folder exists already, just move files there

## MVP

```shell
mediamess organize ./
```
